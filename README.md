### Installation (이 프로젝트를 어떻게 설치하는지에 대한 설명)

- Python == 3.9
```shell
$ pip install -r requirements.txt
```
(파이썬은 몇버전, library 사용한것들은 몇버전, jupyter notbook을 썼다면 그것도 몇버전인지. requirements.txt 여기다가 다 적어줘야함. )
